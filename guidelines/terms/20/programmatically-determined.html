determinado programaticamente (determinável programaticamente)

determinado por software a partir de dados fornecidos pelo autor de forma que diferentes agentes de utilizador, incluindo tecnologias de apoio, possam extrair e apresentar essa informação aos utilizadores em diferentes modalidades

Determinado numa linguagem de marcação a partir de elementos e atributos acedidos diretamente por tecnologias de apoio amplamente disponíveis.

Determinado a partir de estruturas de dados específicas de uma tecnologia numa linguagem não baseada em marcação e disponibilizado às tecnologias de apoio através de uma API de acessibilidade suportada por tecnologias de apoio amplamente disponíveis.
