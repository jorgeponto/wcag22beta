versão alternativa conforme

versão que

  1. está conforme ao nível designado; e
  2. fornece toda a mesma informação e funcionalidade no mesmo idioma humano; e
  3. está tão atualizada quanto o conteúdo não conforme; e

  4. para a qual pelo menos uma das seguintes condições é verdadeira:

    1. a versão conforme pode ser alcançada a partir da página não conforme através de um mecanismo suportado em termos de acessibilidade; ou
    2. a versão não conforme só pode ser alcançada a partir da versão conforme; ou
    3. a versão não conforme só pode ser alcançada a partir de uma página conforme que também disponibiliza um mecanismo para alcançar a versão conforme

Nesta definição, "só pode ser alcançada" significa que existe algum mecanismo, como um redirecionamento condicional, que impede o utilizador de "alcançar" (carregar) a página não conforme, a menos que o utilizador tenha acabado de vir da versão conforme.

A versão alternativa não precisa de corresponder página a página com a original (por exemplo, a versão alternativa conforme pode consistir em várias páginas).

Se estiverem disponíveis várias versões linguísticas, são necessárias versões alternativas conformes para cada idioma disponibilizado.

As versões alternativas podem ser disponibilizadas para acomodar diferentes ambientes tecnológicos ou grupos de utilizadores. Cada versão deve ser tão conforme quanto possível. Uma das versões teria de estar totalmente conforme para satisfazer o requisito de conformidade 1.

A versão alternativa conforme não precisa de estar dentro do âmbito da conformidade, nem sequer no mesmo sítio Web, desde que esteja tão livremente disponível quanto a versão não conforme.

As versões alternativas não devem ser confundidas com conteúdo suplementar, que apoia a página original e melhora a compreensão.

Definir preferências do utilizador no conteúdo para produzir uma versão conforme é um mecanismo aceitável para alcançar outra versão, desde que o método utilizado para definir as preferências seja suportado em termos de acessibilidade.
