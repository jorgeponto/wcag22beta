página Web

um recurso não incorporado obtido a partir de um único URI usando HTTP, juntamente com quaisquer outros recursos utilizados na renderização ou destinados a serem renderizados em conjunto com ele por um agente de utilizador

Embora quaisquer “outros recursos” sejam renderizados em conjunto com o recurso principal, não serão necessariamente renderizados simultaneamente entre si.

Para efeitos de conformidade com estas diretrizes, um recurso deve ser “não incorporado” no âmbito da conformidade para ser considerado uma página Web.

Um recurso Web incluindo todas as imagens e multimédia incorporados.

Um programa de correio eletrónico Web construído com JavaScript assíncrono e XML (AJAX). O programa existe inteiramente em http://example.com/mail, mas inclui uma caixa de entrada, uma área de contactos e um calendário. São fornecidos links ou botões que fazem com que a caixa de entrada, contactos ou calendário sejam apresentados, mas não alteram o URI da página como um todo.

Um portal Web personalizável, onde os utilizadores podem escolher conteúdos a apresentar a partir de um conjunto de diferentes módulos de conteúdo.

Quando introduz “http://shopping.example.com/” no seu navegador, entra num ambiente de compras interativo semelhante a um filme, onde se move visualmente numa loja, arrastando produtos das prateleiras para um carrinho de compras visual à sua frente. Clicar num produto faz com que este seja demonstrado com uma folha de especificações a flutuar ao lado. Isto pode ser um único site de página ou apenas uma página dentro de um site.
