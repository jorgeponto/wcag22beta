interface de teclado

interface utilizada pelo software para obter introdução através de teclas

Uma interface de teclado permite aos utilizadores fornecer introdução por teclas aos programas, mesmo que a tecnologia nativa não contenha um teclado.

Um PDA com ecrã tátil possui uma interface de teclado incorporada no sistema operativo, bem como um conector para teclados externos. As aplicações no PDA podem utilizar a interface para obter introdução por teclado tanto de um teclado externo como de outras aplicações que fornecem saída simulada de teclado, como interpretadores de escrita manual ou aplicações de reconhecimento de voz para texto com funcionalidade de "emulação de teclado".

A operação da aplicação (ou de partes da aplicação) através de um emulador de rato operado por teclado, como MouseKeys, não é considerada operação através de uma interface de teclado, porque a operação do programa é efetuada através da interface de dispositivo apontador e não através da interface de teclado.
