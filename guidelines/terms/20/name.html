nome

texto através do qual o software consegue identificar um componente no conteúdo Web para o utilizador

O nome pode estar oculto e apenas ser disponibilizado através de tecnologia de apoio, enquanto uma etiqueta é apresentada a todos os utilizadores. Em muitos casos (mas não em todos), a etiqueta e o nome são iguais.

Isto não está relacionado com o atributo name em HTML.
