componente de interface de utilizador

uma parte do conteúdo que é percecionada pelos utilizadores como um único controlo para uma função distinta

Múltiplos componentes de interface de utilizador podem ser implementados como um único elemento programático. "Componentes" aqui não está ligado a técnicas de programação, mas sim ao que o utilizador percebe como controlos separados.

Os componentes de interface de utilizador incluem elementos de formulário e ligações, bem como componentes gerados por scripts.

O que aqui se entende por "componente" ou "componente de interface de utilizador" é também por vezes designado "elemento de interface de utilizador".

Um applet tem um "controlo" que pode ser utilizado para navegar no conteúdo por linha, página ou acesso aleatório. Uma vez que cada um destes teria de ter um nome e ser configurável de forma independente, cada um seria um "componente de interface de utilizador."
