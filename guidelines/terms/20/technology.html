tecnologia (conteúdo Web)

mecanismo para codificar instruções a serem renderizadas, reproduzidas ou executadas por agentes de utilizador

Tal como utilizado nestas diretrizes, “tecnologia Web” e a palavra “tecnologia” (quando usada isoladamente) referem-se a tecnologias de conteúdo Web.

As tecnologias de conteúdo Web podem incluir linguagens de marcação, formatos de dados ou linguagens de programação que os autores podem usar isoladamente ou em combinação para criar experiências para o utilizador final que vão desde páginas Web estáticas a apresentações multimédia sincronizadas ou aplicações Web dinâmicas.

Alguns exemplos comuns de tecnologias de conteúdo Web incluem HTML, CSS, SVG, PNG, PDF, Flash e JavaScript.
