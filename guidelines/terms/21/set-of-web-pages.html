conjunto de páginas Web

coleção de páginas Web que partilham um objetivo comum e que são criadas pelo mesmo autor, grupo ou organização

Exemplos incluem:

  • uma publicação dividida por várias páginas Web, em que cada página contém um capítulo ou outra secção significativa da obra. A publicação é logicamente uma unidade contínua única e contém funcionalidades de navegação que permitem o acesso ao conjunto completo de páginas.
  • um sítio de comércio eletrónico apresenta produtos num conjunto de páginas Web que partilham a mesma navegação e identificação. No entanto, ao avançar para o processo de checkout, o modelo muda; a navegação e outros elementos são removidos, pelo que as páginas nesse processo são funcional e visualmente diferentes. As páginas de checkout não fazem parte do conjunto de páginas de produtos.
  • um blogue num subdomínio (por exemplo, blog.example.com) que tem uma navegação diferente e é criado por um conjunto distinto de pessoas em relação às páginas do domínio principal (example.com).

Versões em diferentes línguas são consideradas conjuntos de páginas Web diferentes.
