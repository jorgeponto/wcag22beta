pixel CSS

ângulo visual de cerca de 0.0213 graus

Um pixel CSS é a unidade canónica de medida para todos os comprimentos e medições em CSS. Esta unidade é independente da densidade e distinta dos píxeis físicos existentes num ecrã. Os agentes de utilizador e sistemas operativos devem garantir que um pixel CSS é definido o mais próximo possível do pixel de referência do módulo CSS Values and Units Level 3 [[css3-values]], que tem em conta as dimensões físicas do ecrã e a distância de visualização assumida (fatores que não podem ser determinados pelos autores do conteúdo).
