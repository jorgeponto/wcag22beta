O conteúdo pode ser apresentado sem perda de informação ou funcionalidade e sem exigir deslocamento em duas dimensões para:

  • Conteúdo com deslocamento vertical numa largura equivalente a 320 pixéis CSS;
  • Conteúdo com deslocamento horizontal numa altura equivalente a 256 pixéis CSS.

Exceto para partes do conteúdo que exijam uma disposição bidimensional para utilização ou compreensão.

320 pixéis CSS equivalem a uma largura inicial da viewport de 1280 pixéis CSS com um zoom de 400%. Para conteúdo Web concebido para deslocamento horizontal (por exemplo, com texto vertical), 256 pixéis CSS equivalem a uma altura inicial da viewport de 1024 pixéis CSS com um zoom de 400%.

Exemplos de conteúdo que exige uma disposição bidimensional incluem imagens necessárias para compreensão (como mapas e diagramas), vídeo, jogos, apresentações, tabelas de dados (não células individuais) e interfaces em que é necessário manter barras de ferramentas visíveis durante a manipulação do conteúdo. É aceitável disponibilizar deslocamento bidimensional para essas partes do conteúdo.
