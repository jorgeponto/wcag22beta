Atalhos de Teclas de Carateres

A

Se for implementado no conteúdo um atalho de teclado utilizando apenas letras (incluindo letras maiúsculas e minúsculas), pontuação, números ou símbolos, então pelo menos uma das seguintes condições é verdadeira:

Desativar
Está disponível um mecanismo para desativar o atalho;
Remapear
Está disponível um mecanismo para remapear o atalho de forma a incluir uma ou mais teclas não imprimíveis (por exemplo, Ctrl, Alt);
Ativo apenas com foco
O atalho de teclado de um componente de interface do utilizador só está ativo quando esse componente tem o foco.
