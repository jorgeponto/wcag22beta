A apresentação visual dos seguintes elementos possui uma relação de contraste de pelo menos 3:1 em relação às cores adjacentes:

Componentes da Interface do Utilizador
Informação visual necessária para identificar componentes da interface do utilizador e estados, exceto no caso de componentes inativos ou quando a aparência do componente é determinada pelo agente de utilizador e não modificada pelo autor;
Objetos Gráficos
Partes de gráficos necessárias para compreender o conteúdo, exceto quando uma apresentação específica dos gráficos é essencial para a informação transmitida.
