Identificar a Finalidade da Introdução de Dados

AA

A finalidade de cada campo de introdução de dados que recolhe informação sobre o utilizador pode ser determinada programaticamente quando:

  • O campo de introdução de dados serve uma finalidade identificada na secção Finalidades de Introdução de Dados para Componentes da Interface do Utilizador; e
  • O conteúdo é implementado utilizando tecnologias com suporte para identificar o significado esperado dos dados introduzidos em formulários.
