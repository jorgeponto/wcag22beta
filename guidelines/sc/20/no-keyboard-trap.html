Sem Armadilha de Teclado

A

Se o foco do teclado puder ser movido para um componente da página utilizando uma interface de teclado, então o foco também pode ser retirado desse componente utilizando apenas uma interface de teclado e, se isso exigir mais do que as teclas de seta ou de tabulação sem modificadores ou outros métodos de saída padrão, o utilizador é informado sobre o método para retirar o foco.

Uma vez que qualquer conteúdo que não cumpra este critério de sucesso pode interferir com a capacidade do utilizador de usar toda a página, todo o conteúdo da página Web (quer seja utilizado ou não para cumprir outros critérios de sucesso) tem de cumprir este critério de sucesso. Consulte Requisito de Conformidade 5: Não Interferência.
