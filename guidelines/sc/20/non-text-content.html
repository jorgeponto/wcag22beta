Conteúdo Não Texto

A

Todo o conteúdo não texto apresentado ao utilizador possui uma alternativa em texto que serve a finalidade equivalente, exceto nas situações listadas abaixo.

Controlos, Introdução de Dados

Se o conteúdo não texto for um controlo ou aceitar introdução de dados pelo utilizador, então possui um nome que descreve a sua finalidade. (Consulte o Critério de Sucesso 4.1.2 para requisitos adicionais relativos a controlos e conteúdo que aceita introdução de dados pelo utilizador.)

Multimédia Baseada no Tempo

Se o conteúdo não texto for multimédia baseada no tempo, então as alternativas em texto fornecem, pelo menos, uma identificação descritiva do conteúdo não texto. (Consulte a Diretriz 1.2 para requisitos adicionais relativos a multimédia.)

Teste

Se o conteúdo não texto for um teste ou exercício que seria inválido se apresentado em texto, então as alternativas em texto fornecem, pelo menos, uma identificação descritiva do conteúdo não textual.

Sensorial

Se o conteúdo não texto se destinar principalmente a criar uma experiência sensorial específica, então as alternativas em texto fornecem, pelo menos, uma identificação descritiva do conteúdo não texto.

CAPTCHA

Se a finalidade do conteúdo não texto for confirmar que o conteúdo está a ser acedido por uma pessoa e não por um computador, então são fornecidas alternativas em texto que identificam e descrevem a finalidade do conteúdo não texto, bem como formas alternativas de CAPTCHA que utilizem modos de saída para diferentes tipos de perceção sensorial, de forma a acomodar diferentes incapacidades.

Decoração, Formatação, Invisível

Se o conteúdo não texto for puramente decorativo, for utilizado apenas para formatação visual ou não for apresentado aos utilizadores, então é implementado de forma a poder ser ignorado pela tecnologia de apoio.
