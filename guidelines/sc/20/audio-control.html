Controlo de Áudio

A

Se algum áudio numa página Web for reproduzido automaticamente durante mais de 3 segundos, está disponível um mecanismo para pausar ou parar o áudio, ou está disponível um mecanismo para controlar o volume do áudio independentemente do nível de volume geral do sistema.

Uma vez que qualquer conteúdo que não cumpra este critério de sucesso pode interferir com a capacidade do utilizador de usar toda a página, todo o conteúdo da página Web (quer seja utilizado ou não para cumprir outros critérios de sucesso) tem de cumprir este critério de sucesso. Consulte Requisito de Conformidade 5: Não Interferência.
