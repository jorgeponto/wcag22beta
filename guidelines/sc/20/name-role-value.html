Nome, Função, Valor

A

Para todos os componentes da interface do utilizador (incluindo, entre outros, elementos de formulários, ligações e componentes gerados por scripts), o nome e a função podem ser determinados programaticamente; os estados, propriedades e valores que podem ser definidos pelo utilizador podem ser definidos programaticamente; e a notificação de alterações destes itens está disponível para os agentes de utilizador, incluindo as tecnologias de apoio.

Este critério de sucesso destina-se principalmente a autores Web que desenvolvem ou criam por script os seus próprios componentes da interface do utilizador. Por exemplo, os controlos HTML padrão já cumprem este critério de sucesso quando utilizados de acordo com a especificação.
